    上海瑞宙生物科技有限公司是由“千人计划”特聘归国专家领衔的高新生物制药及生物技术企业。公司专注于开发高附价值、高质量生物技术药物和试剂产品。

    以提高人类健康为宗旨，服务社会为基石，本着中西融合、厚德载物、行胜于言理念，在相关政府和专家的关心和支持下，公司有效整合多方资源，造就一支专业敬业团队，开展从基因工程、发酵、纯化、免疫及质量分析等方面的系统性研究，开发高质量的重组蛋白类药物、新型疫苗及相关生物试剂产品。
